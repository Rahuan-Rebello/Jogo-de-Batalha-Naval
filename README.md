# 🚢 Batalha Naval - Java Edition

Uma implementação do clássico jogo de estratégia **Batalha Naval**, desenvolvida em **Java**. Este projeto foca na aplicação de conceitos fundamentais de Programação Orientada a Objetos (POO) e lógica de matrizes.

## 📌 Sobre o Projeto

O jogo é executado via console, onde o jogador compete contra uma IA para descobrir e afundar a frota inimiga em um tabuleiro 10x10. O projeto foi estruturado para ser modular, facilitando a manutenção e a expansão de funcionalidades.

### ✨ Funcionalidades

* **IA de Posicionamento:** O computador posiciona as embarcações de forma aleatória e inteligente.
* **Validação de Jogadas:** Sistema que impede tiros repetidos ou fora dos limites do tabuleiro.
* **Interface via Console:** Representação visual do tabuleiro utilizando caracteres ASCII para indicar acertos, erros e embarcações.
* **Controle de Estados:** Gestão de turnos e verificação automática de condições de vitória.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java 17+
* **Conceitos de POO:** Encapsulamento, abstração e separação de responsabilidades.
* **Estrutura de Dados:** Matrizes bidimensionais para representação do campo de batalha.
* **Entrada/Saída:** `java.util.Scanner` para interação via terminal.
